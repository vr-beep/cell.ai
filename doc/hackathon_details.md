## About the Event

For the fifth hack in this series, we're hosting another cell culture hack that brings together the Bay Area's AI, science, and engineering communities. Teams will have access to the latest AI tools for building agents and are given the opportunity to ship on Monomer's robotic workcell and individual walk-up instruments.

**No pre-reqs.** All scientists, product designers, software, and hardware engineers welcome. All skill levels are welcome and the event can support 50 builders.

> **Important:** You must be able to stay through demo on Sunday to participate in the hack. Everyone joining will be expected to demo something by end of the weekend.

---

## Hackathon Tracks

### Track A: Develop an AI Agent to Optimize Media Composition (Closed Loop)

Build an autonomous workflow that iteratively varies media components, runs cell culture, measures outcomes, and updates the next experiment end-to-end. All on REAL cells.

**Robotic workcell actions available via MCP:**
- 🌡️ Incubate
- 🧪 Create media mixture
- 💧 Media change
- 🔬 Plateread

### Track B: Use AI Coding Tools to Develop Protocols

Program hitpicking and imaging methods on the following instruments:
- **Liquid handler:** Opentrons OT-2 running PyLabRobot
- **Cephla Microscope**
- More to announce later

### Track C: Implement Physical AI to Automate Flask-Based Cell Culture

Program a robotic arm to manipulate manual instruments and automate large volume cell culture.

**Hardware available:**
- Manipulate manual incubator
- Manual microscope
- T-Flasks
- Serological pipettes and tips
- Automated bottle opener
- Robotic arm
